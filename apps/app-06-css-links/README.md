# CSS Links Demo

A minimal Angular project that demonstrates **styling hyperlinks** with CSS — showing default link appearance, hover/focus states, and the difference between external `<a href>` and Angular's internal navigation with `[routerLink]`.

### What it demonstrates

- Basic link styling (`color`)
- Interactive states: `:hover` and `:focus` (using `opacity` for subtle feedback)
- External links (`target="_blank"`)
- Internal Angular routing links (`[routerLink]`)
- How to create accessible and visually clear link interactions without changing color (common modern pattern)
- Commented-out color change on hover/focus for easy comparison
